# free-rein-pilates

A modern website for Free Rein Pilates studio showcasing classes, instructors, and wellness philosophy

## Created with Bolt Smart

- **Created**: 2025-05-30T23:04:43.690Z
- **React**: 18.3.1
- **TypeScript**: 5.4.0
- **Vite**: 5.4.2
- **Tailwind CSS**: 3.4.0

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Features

- ⚡️ Lightning fast with Vite
